# 에셋·산출물 목록 (MVP)

구현과 배포(로컬)에 필요한 산출물을 정리했습니다.

## 필수

| 항목 | 설명 |
|------|------|
| `.env.example` | LLM/API 관련 변수명과 설명(실제 비밀값 없음) |
| `README.md` | OS별 선행 요건, 설치, `localhost` 실행, 트러블슈팅 |
| 샘플 입력 | 스모크용 짧은 `.txt`, 1~2페이지 수준 `.pdf` |
| 아이콘/파비콘 | 선택 — 없어도 MVP 성립 |

## 선택

| 항목 | 설명 |
|------|------|
| 간단한 로고 또는 앱 이름 확정 | UI 헤더·창 제목 |
| 프롬프트 템플릿 파일 | 요약·Q&A 시스템 프롬프트를 코드와 분리해 버전 관리 |
| 라이선스 표기 | 사용하는 PDF 라이브러리·폰트·아이콘 세트 |

## 데이터·저장

| 항목 | 설명 |
|------|------|
| 로컬 데이터 디렉터리 | 예: `./data` 또는 OS별 앱 데이터 경로 — `.gitignore`에 포함 |
| 세션 스키마 | 문서 ID, 생성 시각, 제목(파일명), 마지막 질문 시각 등 최소 필드 |

## 검증용

| 항목 | 설명 |
|------|------|
| 스모크 체크리스트 | `docs/TASKS.md` T14와 연동 |
| (선택) 스크린 녹화 또는 스크린샷 | 온보딩용 |
